##所需软件

- **ffmpeg**

        sudo apt-get install ffmpeg

- **mp3splt (不是 split)**

        sudo apt-get install mp3splt

##具体步骤

1. 首先将 APE 转换成 MP3 格式

        ffmpeg -i src.ape dst.mp3

2. 根据 CUE 文件将第一步中生成的 MP3 文件切割

        mp3splt -c yours.cue dst.mp3
